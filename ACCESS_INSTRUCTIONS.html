Access Ampere Engineering Web App

Step 1: Open Your Browser

Open your preferred web browser (Chrome, Firefox, Edge, etc.)

Step 2: Navigate to the Application

Enter the following URL in your browser's address bar:

http://localhost:3004

Step 3: Login to the Application

Use one of the following credentials to log in:

  • Super Administrator: admin@ampere.com / password
  • Administrator: manager@ampere.com / password
  • Finance Team: finance@ampere.com / password
  • Project Manager: pm@ampere.com / password

Important Notes:

  • The development server is currently running in the background
  • If you restart your computer, you'll need to start the server again using: npm run dev
  • If port 3004 is already in use, the application will automatically try the next available port
  • All the purchase order functionality you requested is now implemented and accessible

Need to Restart the Server?

If you need to restart the development server, open a terminal in the project directory and run:

npm run dev
